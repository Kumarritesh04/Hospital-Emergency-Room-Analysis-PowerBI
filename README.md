# Hospital-Emergency-Room-Analysis-PowerBI

## Project Overview
This Power BI dashboard analyzes Hospital Emergency Room data to provide insights into patient flow, waiting time, satisfaction scores, department referrals, and demographic trends.

## Dashboard Highlights

- Total Patients: 9K+
- Average Waiting Time Analysis
- Patient Satisfaction Monitoring
- Department Referral Analysis
- Gender Distribution Analysis
- Interactive Filtering and Drill-Down Capabilities

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key Insights
- Majority of patients did not require department referrals.
- General Practice and Orthopedics received the highest referrals.
- Patient satisfaction trends were monitored across departments.
- Gender distribution of patients was nearly balanced.
